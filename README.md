# 🎮 המשחקים של המשפחה

שני משחקי לימוד עצמאיים (HTML/CSS/JS, ללא תלויות, ללא build) — כל אחד עובד גם ישירות מהדפדפן (`open index.html`) וגם דרך שרת.

## המשחקים

- **[מִילִימוֹן](milimon/index.html)** 🎴 — תרגול קריאה בעברית בסגנון קלפי פוקימון, לגיל 5–7.
- **[מסע היהלומים](gemquest/index.html)** 💎 — חשבון והבנת הנקרא לכיתה ה׳.

## הפעלה מקומית

לחיצה כפולה על **`הפעל שרת משחקים.command`** — פותח שרת מקומי ומציג כתובת לשיתוף עם מכשירים אחרים באותו WiFi (טלפון, אייפד, מחשב נוסף).

## פרטים טכניים

ראו `PROJECT_SUMMARY.html` בתוך כל תיקיית משחק לארכיטקטורה מלאה, החלטות עיצוב והוראות תחזוקה.
